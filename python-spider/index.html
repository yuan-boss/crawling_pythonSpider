凯酷KC818 RGB机械键盘拆解评测

神秘大全

百家号 03-04 22:17

凯酷最近众筹了一款新机械键盘——凯酷KC818，该键盘做了全新的模具设计，这款键盘有诸多亮点，搭载炫酷的RGB灯光系统，配套搭配了RGB手托，在键盘侧翼还做了侧翼灯光设计，还有音量调节旋钮，整体颜值相当不错。

整体外观

凯酷KC818 RGB机械键盘包装比较简洁，包装正面右侧做了透明的开口设计，可以透过开头的隔层透明塑料，看到键盘的样貌。包装背面则是产品的参数说明以及厂家的信息。

产品的主要配件：凯酷KC818 RGB机械键盘、合格证、拔键器、说明书和键帽布罩。

凯酷KC818 RGB机械键盘采用标准美式104键配列，键盘框架为目前比较流行的金属一体式面板和悬浮式按键设计，键盘自带配套的手托（可拆解）所以键盘会显得比较宽大，卸下手托就我们常规的104键的机械键盘类似。

规格参数：

轴开关：佳达隆

按键数：104键

规格：446.4 x 195.3 x 40mm（含手托）

重量：1.2Kg

灯光：RGB

凯酷KC818 RGB机械键盘采用全黑配色，加上手托后，看起来相当有霸气，手托上白色灯带和LOGO灯位显得格外醒目。键盘面板右上角采用状态灯光加音量调节滚轮的设置，值得赞扬。

拆掉手托后，就明显干练了很多，不过也少了几分特色。

键盘底部设计平整，中间有阶梯状对称的纹路，极具现代感，键盘四周各有一个条形脚贴。手托部分分布3个脚贴，通过卡扣和键盘连接。发光部分的电源则是通过排线和键盘连接，如果做成强力磁铁和针脚接触，那就完美了。

细节做工

凯酷KC818 RGB机械键盘配套的手托做工还是比较不错的，表面为黑色磨砂质感，手感舒适，另外发光不做采用透明塑料材质做嵌入式设计，底部为白色背景，主要起到一个导光作用。另外LOGO部分也做了透光处理，通电后B格非常高。

金属上盖部分表面采用细腻的喷砂工艺，摸起来有类似细细的砂纸的感觉，手感还不错，不过你用指甲或者键帽划过表面的话，会留下白色的痕迹，好在这个痕迹可以擦拭，这种表面还是比较耐用的。键盘上盖上延做了上扬的弯曲设计，颇有个性，且边缘做了抛光处理。

凯酷KC818 RGB机械键盘金属面板右上角的布局亮点十足，有四个状态灯光位，状态标识采用镀银色，右侧的音量调节旋钮非常有质感，表层为金属面，做了网状纹理，滚动起来比较顺畅，有轻微阻尼感。

键盘配套的手托与键盘的坡度贴合度很高，使用起来可以很好的缓解我们手腕的压力。

键盘左右侧翼均有闪电状态的发光条，在通电后，会有不错的灯光效果，而且最重要的是灯光颜色还可以调节。

键盘撑脚设计比较厚实，且撑脚顶端有做防滑橡胶层。

凯酷KC818 RGB机械键盘采用黑色编织数据线，由于多了个手托发光，为了满足供电量，所以线材比较粗，线材质感不错，线上配有理线带、抗干扰磁环。

使用体验

凯酷KC818 RGB机械键盘驱动界面非常简练，左侧是功能选项，如模式选择，灯光设置以及恢复出厂设置选项，右侧则是键盘按键设置和回报率设置，上手操作简单。

点击驱动界面键盘上的按键，即可对单个按键进行设置，功能丰富，还可以直接分配宏命令，非常实用。宏录制界面可以直接修改按键命令、动作和延迟时间。

凯酷KC818 RGB机械键盘预设的灯光效果有19种之多，玩家可以自定义灯光颜色，而且每种效果也可以单独设置颜色，可玩性相当高，操作起来也比较。

凯酷KC818 RGB机械键盘搭配的键帽和公模键帽接近，部分字体有了些变动，键帽字体清晰，辨识度高，常规的104配列也比较容易上手。

凯酷KC818 RGB机械键盘的键帽为ABS二色键帽，表面为磨砂质感，手感细腻舒适，耐用度也非常不错，键帽边缘打磨的精细，无明显的毛刺感。

凯酷KC818 RGB机械键盘搭载佳达隆轴体，佳达隆轴目前在国产品牌机械键盘中运用还是比较普遍，轴体手感和原厂轴类似，该把评测键盘的轴体是佳达隆青轴，手感清脆，段落感清晰，稳定性也非常不错，大键位部分采用卫星轴设计，手感调校的也非常出色。

出色的灯光效果也是凯酷KC818 RGB机械键盘的最强卖点之一，首创RGB幻彩手托给这款键盘增色不少，闪电纹路的侧翼灯光可以独立调节颜色。该款键盘预设的灯光效果非常多，

拆解部分

凯酷KC818 RGB机械键盘拆解比较简单，只要拆除键盘上盖上的十几个螺丝（如图所示)，即可轻松分离上下壳。

键盘塑料底壳材质厚实，做工也比较精致，无明显毛刺感。

键盘底壳右上角集成了音量调节旋钮，其原理和我们鼠标滚轮类似，并且滚轮右侧还配有滚轮灯。

凯酷KC818 RGB机械键盘采用蓝色PCB板，板面电子元件较多，排列整齐，轴和元件焊点饱满，做工比较出色。

凯酷KC818 RGB机械键盘侧翼灯光通过PCB板边缘的LED直射，这几个LED通过手工焊接，所以看起来做工会比较粗糙。

键盘前端手托供电针脚。

键盘主控为VS11K09A，目前国产RGB机械键盘中比较常见。

键盘的钢板和PCB板用料厚实。

总结

凯 酷KC818 RGB机械键盘从设计到体验，都看出了凯酷在进步，在满是抄袭的外设市场下，还有自己的新创意出来，这点就值得点赞。凯酷KC818 RGB机械键盘灯光效果非常出色，特别是首创的RGB幻彩手托，给这款键盘增色不少，另外闪电状可调节颜色的侧翼灯光，简洁的键盘驱动，也增加了这款键盘 的可玩性，搭载佳达隆轴体，手感也有不错的表现。不过这款键盘还是有提升的空间，比如手托连接的部分，还有手托和侧翼灯光还没集成到驱动中等。后面该款键 盘还会推出KC828樱桃轴版本，喜欢的朋友关注下凯酷官方网站。
